TDD Overview

  • Test-driven development starts with developing tests for each one of the features.
  • The test might fail as the tests are developed even before the devleopment
  • The development team develops and refactors the code to pass the tests
  • Test-driven development is related to the test first programming evolved as part of extreme programming concepts

Test-Driven Development Process

  • Add a test
  • Run All test and see if the new one failes
  • Write some code
  • Run tests and refactor code
  • Repeat
